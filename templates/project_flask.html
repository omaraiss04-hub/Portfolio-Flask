Flask Portfolio Backend

Python 3.13 Flask Framework REST-API Asynchronous (AJAX) JSON

Projektbeschreibung

Dieses Projekt bildet (das Rückgrat) für mein digitales Portfolio. Es handelt sich um ein leichtgewichtiges, aber hochperformantes Backend, das auf dem Python-Flask-Framework basiert. Das Hauptziel war es, eine moderne Architektur zu schaffen, die Benutzeranfragen blitzschnell und ohne spürbare Verzögerung verarbeitet.

Technische Kernfunktionen

  • Asynchrones Kontaktformular: Durch den Einsatz der JavaScript Fetch API werden Formulardaten im JSON-Format an das Backend gesendet, ohne dass die gesamte Webseite neu geladen werden muss (No-Page-Reload).
  • Echtzeit-Terminal-Logging: Eingehende Nachrichten werden strukturiert und formatiert direkt im VS Code Terminal ausgegeben, was ein optimales Live-Monitoring ermöglicht.
  • Robuste Datenvalidierung: Das Flask-Backend prüft jede Anfrage serverseitig auf Vollständigkeit (Schutz vor leeren oder fehlerhaften Datensätzen) und antwortet mit standardisierten HTTP-Statuscodes (z. B. 400 Bad Request oder 200 OK).
  • Modularer Code: Die Ordnerstruktur trennt strikt zwischen Logik (Python), Präsentation (HTML/Templates) und statischen Assets (CSS/Bilder).
